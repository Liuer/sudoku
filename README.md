# sudoku

## how to build
 
```
cd <project-dir>
go build
```
in windows you will see the sudoku.exe at current dir

## start as http serve
```
./sudoku.exe -addr :8080
```
it will listen 8080 port 
```
// use curl 
curl -i -X POST \
```
